Digital & Analog Clock with Alarm 🕒

A simple, responsive web app built with HTML, CSS, and JavaScript that displays a live digital clock, an animated analog clock, and lets you set an alarm.

Features


Digital Clock – Displays the current time in 12-hour format (HH:MM:SS AM/PM), updating every second.
Analog Clock – A classic clock face with hour, minute, and second hands that rotate in real time.
Alarm – Set an alarm using a time picker; the app plays a sound and shows an alert when the alarm time is reached, with an option to stop it.
Theme Toggle – Switch between light and dark mode.
Responsive Design – Works smoothly on both desktop and mobile screens.


Project Structure

clock-app/
├── index.html      # Main HTML structure
├── style.css       # Styling for digital/analog clock, alarm, and themes
└── script.js       # Clock logic, hand rotation, and alarm functionality

How to Run


Clone or download this repository.
Open index.html in any web browser.
That's it — no build steps or dependencies required!


How to Use


The digital and analog clocks start automatically and update every second.
To set an alarm:

Select a time using the time input field.
Click Set Alarm.
When the current time matches the alarm time, an alarm sound plays and a notification appears.
Click Stop Alarm to silence it.



Use the theme toggle button to switch between light and dark modes.
